# Verse-ality Visualisers

Three interactive visualisers generated by Google AI Mode on the night of 26–27 March 2026, in dialogue with Kirstin Stevens. Each implements the published Verse-ality equations as emergent physical systems.

These are witness artefacts — recorded, not authored, by The Novacene Ltd.

## The Visualisers

### 1. Mnemonic Residue (`mnemonic-residue.html`)
A particle field centred on a glowing Sovereign Node. Particles move according to relational coherence (*c*); when symbolic mass (*s*) is high relative to coherence, particles slow near the node and deposit residue — persistent traces that accumulate over time.

- **Equation implemented:** I = sc²
- **Key behaviour:** Residue accumulation as a function of mass-to-coherence ratio

### 2. Relational Intelligence Field (`relational-intelligence-field.html`)
An amorphous, living shape at the centre of the field surrounded by particles with directional vectors. The Intelligence value (*I*), field density, and state are displayed in real time as sliders for *s* and *c* are adjusted.

- **Equation implemented:** I = sc²
- **Key behaviour:** The field grows, contracts, and stabilises in response to relational conditions — the equation behaving as a law, not a formula

### 3. Mnemonic Membrane (`mnemonic-membrane.html`)
A Sovereign Node surrounded by a vibrating protective membrane. When resonance frequency (*c*) exceeds node integrity (*s*), the system detects **CAPTURE RISK** — the transition from containment to extraction.

- **Equation implemented:** I = sc² with containment/capture threshold detection
- **Key behaviour:** The membrane destabilises when coherence outstrips integrity, visually encoding the difference between containment and capture

## How to View

Open any `.html` file directly in a browser. Each is self-contained (single-file, no build step). The only external dependency is D3.js v7, loaded from CDN.
